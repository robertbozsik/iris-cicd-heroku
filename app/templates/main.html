Machine Learning Model Deployment

The goal of this project was to create an end-to-end data product by deploying a simple machine learning model. I used Flask to create the web application, Docker to containerize it, Travis CI to automate the deployment in case of any changes, and for the second deployment Heroku instead of AWS ECS to store the app.

The model was trained based on the classical Iris dataset created by R. A. Fischer, included in scikit-learn within the dataset module. Go and try it out by adding the length and width parameters, after that hit the submit button in order to figure out what species your iris belongs to.
